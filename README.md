# 🎬 Skynie – Movie Ticket Booking App UI Kit

## 📌 Description
**Skynie** is a modern Movie Ticket Booking **UI Kit** designed for mobile applications. It covers the complete user journey of a cinema booking system, including authentication, browsing movies, selecting seats, and managing tickets.

---

## ✨ Features
- Modern and visually appealing UI  
- Complete booking flow from login to ticket confirmation  
- Fully customizable design  
- Easy color and component modification  
- Designed using Figma  
- Uses free Google Fonts  

---

## 📱 Screens Included
- Splash Screen  
- Onboarding  
- Login / Sign Up  
- Login  
- Sign Up  
- Forgot Password  
- OTP Verification  
- Home  
- Search  
- Film Details  
- Book Tickets  
- Choose Seat  
- Order Details  
- My Tickets  
- Ticket Details  
- Watch Trailer  
- Account
